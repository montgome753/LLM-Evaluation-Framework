Metrics

5 metrics that matter

Every metric is computed asynchronously per sample and aggregated with full percentile stats.

Metric Description How it's computed Output
🎯 Accuracy Is the answer correct? Exact → normalized → fuzzy match cascade 0.0 – 1.0
⚡ Latency How fast is the model? Wall-clock time per call, p50/p95/p99 reported ms
💰 Cost How much does it cost? Token counts × provider pricing table USD / 1K tokens
🤥 Hallucination Is the model making things up? Linguistic signal analysis (no extra API call) 0.0 – 1.0
🧠 Reasoning How well does it reason? Reasoning marker density + grounding signals 1 – 10
REST API

with OpenAPI docs
